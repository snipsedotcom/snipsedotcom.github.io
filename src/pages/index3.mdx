# Markdown Features

Docusaurus supports **[Markdown](https://daringfireball.net/projects/markdown/syntax)** and a few **additional features**.

export const year = 2022
export const asdasdpl = 'world'

# Hello {asdasdpl.toUpperCase()}



export const Year = () => (<span> {year} {asdasdpl.toUpperCase()} </span>);

export const Content = ({content}) => (<span> {content} </span>);
